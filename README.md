# Marble Logo

This folder contains the logo assets for Marble, a platform to discover and share designs for digital products.

## Files Included

- **marble-logo.svg**: The main detailed logo with marble-like texture and colored veins
- **marble-logo-simple.svg**: A simplified version that works well at smaller sizes
- **marble-logo.html**: An HTML file that displays the logo with description

## About the Logo Design

The logo represents the concept of marble - a beautiful stone that forms over time by compressing many different rocks into one cohesive material. This mirrors Marble's platform purpose: bringing together diverse designs and creating a beautiful shared space for digital product design.

### Design Elements

1. **Circular Shape**: Represents unity and completeness
2. **Gradient Background**: Represents the smooth texture of polished marble
3. **Colored Veins**: The distinctive blue and purple veins represent the variety of designs flowing through the platform
4. **Simple Typography**: Clean, modern font treatment for the name

## Usage

To view the logo, open the HTML file in a web browser:

```
open marble-logo.html
```

The SVG files can be used directly in websites, apps, or printed materials. They are vector-based and will scale to any size without losing quality.

## Color Palette

- Light gray-blue: #f8f9fa to #adb5bd (marble base)
- Accent blue: #4895ef (veins)
- Accent purple: #7209b7 (veins)
- Text color: #495057
